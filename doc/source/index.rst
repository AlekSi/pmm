|pmm.name| Documentation
********************************************************************************

|percona-monitoring-management| is an open-source platform for managing and
monitoring |mysql| and |mongodb| performance.  It is developed by |percona| in
collaboration with experts in the field of managed database services, support
and consulting.

|pmm| is a free and open-source solution that you can run in your own
environment for maximum security and reliability.  It provides thorough
time-based analysis for |mysql| and |mongodb| servers to ensure that your data
works as efficiently as possible.
